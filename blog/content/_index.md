Lakona is a set of libraries for building online multiplayer games in C# with Unity, Godot, and .NET.

- **[Lakona.Rpc](/rpc/)** — strongly typed bidirectional RPC framework. Start here for networking fundamentals, contracts, source generation, server/client transport, and API reference.
- **[Lakona.Game](/game/)** — actor-based game-session infrastructure built on Lakona.Rpc. Start here for session management, hot-reloadable game logic, reliable push, and server deployment.
