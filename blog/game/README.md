# Lakona.Game Blog

This directory contains the Hugo site for Lakona.Game documentation and blog-style articles.

## Local Usage

```bash
cd blog
hugo server
```

## Build

```bash
cd blog
hugo
```

## GitHub Pages

The site is configured as a repository project site:

- `https://bruce48x.github.io/Lakona.Game/`

If the deployment target changes, update `baseURL` in `blog/hugo.toml`.
